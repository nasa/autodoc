Instructions

  1. Choose a text file as input

  2. text file format must be space-delimited as in key(space)value

  3. Choose a template as input(.docx,.pptx,.xlsx)
  • Mark positions in document like so:
    • ${unique_bookmark_name}
    • OR [unique_bookmark_name]
    • OR unique_bookmark_name
  1. Click generate. Choose where to save the output to


Bookmarks using symbols such as ${unique_bookmark_name} must be stated exactly the same in the text file


Auto-open option opens up the output once generation is complete
